STEPS TO RUN PROJECT LOCALLY
  1)Clone the repo:
    git clone https://github.com/CHARITHAREDDYMAKER/DEWY.git
    cd DEWY
  2)Install dependencies:
    pip install flask scikit-learn tensorflow
  3)Run the backend (Flask):
    cd SkinDiseaseAssistant
    python app.py
  4)Open in Browser:
    http://localhost:5000
